Everyone knows about positive Search Engine Optimization, but what causes negative optimization and what is its impact?

This blog will explain what negative SEO is, what it causes, and the impact it can have.

Google and other search engines visit pages and review their content to determine when to display them when someone is searching for information. They take into account the page's title, text, and semantic HTML tags used. If these things are not in order on a site, it can result in a (drastic) drop in search engine rankings. Below are various examples of what can cause negative SEO.

Title doesn't match the content

Search engines have become so smart that they understand the content of a page. When the page's title, the "H1" title, and the content don't match, it can result in negative SEO.

Missing HTML tags

One of the most important tags that should never be missing from a page is the <h1> tag. For search engines, this is a critical element on a site and should represent the page's content. If this tag and title are missing, a 'crawler' or 'spider,' programs that automatically view a page, will have trouble determining what the page is really about.

Wrong "Document Outline"

Another critical aspect of a page is the "Document Outline," which can be compared to a book:

  • Every book has a title, which is the <H1> tag on the page.
  • The text in a book is divided into chapters, the <H2> tags in the HTML.
  • Under chapters, there can be paragraphs, the <H3> tags in the HTML.
  • Each paragraph that belongs to another paragraph is given a subsequent number: <H4> --> <H5> --> <H6>.

Commonly Used "Tricks"

In the past, it was possible to apply tricks to a page to gain an advantage in search engines. Fortunately, crawlers are getting smarter and can detect these tricks. Below are some of these tricks.

Text color equal to background

Having a lot of content on a page is good for SEO, but often unrelated content was also added. This also resulted in a better position. To make sure visitors didn't see this content, the color of the text was made the same as the background. This made the text invisible to visitors, but visible to crawlers.

Adjusting position

Another way to make unrelated content invisible to visitors was to position this text off-screen. Visitors have no idea that this text exists, but crawlers think it adds value to the page.

Adding tags

Pages also used a list of "tags". The tags were individual words that, like the previous points, gave crawlers the impression that they added value.

All these "tricks" no longer work because crawlers have learned to recognize these techniques and give negative marks on search results.

It is important to consider the content on a site and how this content is displayed. There are many more ways in which a site can contain negative or positive SEO. In the next blog, examples will be given of how a site or page contains positive SEO.
